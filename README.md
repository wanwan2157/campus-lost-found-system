# Campus Lost and Found System

## Project Overview
This project is a campus lost and found management system designed for Lost Owners, Finders, and Administrators. It supports role-based login, real-name authentication, lost/found item reporting, matching, review, notification, and statistical analysis.

## Core Features
- Role-based login and real-name authentication
- Lost item publishing and management
- Found item registration and query code support
- Automatic matching and notification
- Administrator review and claim processing
- Statistics and report export

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Python, FastAPI
- Database: SQLite
- Version Control: GitHub
